# User Management System

A full-stack User Management System developed using React, Node.js, Express.js and MySQL.

## Features

- Add new users
- View all users
- Edit user details
- Delete users
- Search users

## Technologies Used

Frontend:
- React.js
- Axios
- CSS

Backend:
- Node.js
- Express.js

Database:
- MySQL
